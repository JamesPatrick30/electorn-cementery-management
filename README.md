# Vue 3 + TypeScript + Vite Cemetery Management System

A modern desktop application built with Vue 3, TypeScript, and Vite for managing cemetery records and operations.

## Features

- Built with Vue 3 and TypeScript for type safety
- Fast development and build times with Vite
- Electron integration for desktop application
- Component-based architecture with `<script setup>` SFCs

## Getting Started

### Prerequisites
- Node.js 16+ installed

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```
